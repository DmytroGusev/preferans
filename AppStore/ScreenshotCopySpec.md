# Screenshot Copy Spec

Use these as the first-pass App Store caption lines and layout direction.

## Visual Direction

- Tone: premium card room, not casino neon
- Palette: deep green felt, warm ivory cards, brass accents
- Typography: clean serif or humanist sans for authority
- Composition: one clear gameplay action per screenshot
- Avoid: tiny text, cluttered tables, placeholder names, mock URLs

## iPhone Screenshot Set

### 1. Lobby

- Headline: `Classic Preferans, built for iPhone`
- Support line: `Set up a 3 or 4 player table in seconds`
- Show: player count selector, polished player names, start action

### 2. Invite Friends

- Headline: `Invite friends into your table`
- Support line: `Share a join link and start the hand together`
- Show: room code, invite action, clean lobby state
- Note: only use this if live multiplayer actually ships

### 3. Bid For Trump

- Headline: `Bid smart and claim the talon`
- Support line: `Choose the suit that gives you control`
- Show: active bidding controls and best current bid

### 4. Play The Hand

- Headline: `Follow suit. Trump at the right moment.`
- Support line: `Clear touch controls keep every trick readable`
- Show: cards on table, active player, playable hand

### 5. Track The Score

- Headline: `See the hand unfold at a glance`
- Support line: `Monitor tricks, scores, and the current leader`
- Show: end-of-hand or mid-hand scoreboard with strong contrast

## iPad Screenshot Set

Use the same messaging with wider table composition and more breathing room.

## Store Copy Guardrails

- Do not claim real-money play
- Do not claim online multiplayer unless the build truly supports it
- Do not mention bots unless solo AI is implemented
- Keep screenshot text consistent with the shipped ruleset
