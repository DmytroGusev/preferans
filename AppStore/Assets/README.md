# App Store Asset Folder

Put final marketing assets here before submission.

Suggested contents:

- `icon-source/` for layered source files
- `icons-export/` for generated PNG app icons
- `screenshots/iphone/`
- `screenshots/ipad/`
- `promo/` for optional website or press assets

This folder is intentionally documentation-only right now because the project does not yet contain final artwork.
